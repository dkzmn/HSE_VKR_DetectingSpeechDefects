# Эксперименты: детекция дефектов речи

Эксперименты расположены в папках **exp_XX_*/**

 **Подготовка:** перед запуском экспериментов выполните блокнот **`00_dataset_analysis.ipynb`** (в корне `experiments/`). Он загружает/проверяет данные, удаляет пустые аудиофайлы, извлекает из имён файлов буквы скороговорок как признаки и сохраняет **`data/dataset.csv`** с колонками: `filename`, `dir`, `label`, `duration`, `<буквы>`.

Зависимости: `pip install -r experiments/requirements.txt`

В каждой папке:
- **notebook.ipynb** — блокнот для запуска и воспроизведения эксперимента;
- **result.csv.ipynb** — результат;

## Метрики

Каждый эксперимент записывает результат в свой файл **`exp_XX_*/result.csv`**.
Общая таблица результатов (а также топ лучших) - ноутбуке **`01_results_summary.ipynb`**
