# План работы над ВКР

**Тема: Детекция дефектов речи с помощью скороговорок**

**Автор: Кузьмин Дмитрий Андреевич**


## Аналитический обзор предметной области

1. Изучение типов дефектов речи (особенности, признаки)
2. Обзор существующих решений

## Анализ данных

1. Анализ существующего датасета (классы, виды скорговорок)
2. Поиск публичных данных для расширения датасета (логопедические сообщества, приложения)
3. Выбор инструментов обработки аудиофайлов (Librosa, torchaudio)
4. Изучение инструментов для извлечения признаков (MFCC, Mel-спектрограммы, Chroma, VAD, Pitch, jitter, shimmer)
5. Аугментации данных (шумы, изменение темпа, изменение высоты тона, компрессия)
6. Выбор метрик

## Обзор методов машинного обучения и глубинного обучения

1. ML-подходы:

    - SVM, Random Forest, Boostings и другие

2. DL-подходы:

    - CNN на спектрограммах
    - RNN/GRU/LSTM для временных рядов
    - Transformers

3. Fine-tuning предобученных аудио-моделей

4. Выбор финальной модели

## Разработка Telegram-сервиса

1. Написание telegram-бота
    - обработка поступивших аудио/видео файлов, голосовых сообщений
    - разграничение доступа (доступ для наблюдаемых и для логопедов с возможностью просмотра динамики наблюдаемых)
2. Серверная часть
    - API для обработки аудио
    - inference лучшей модели
3. Развёртывание
    - Docker
    - Хостинг
    - Логи и мониторинг
